	- mag_z

	- external shears
	- position angles of external shears
	- image_g
	- image_r
	- image_z

	- Y2 (arcsec)

	- mag_r of QSOs
	- mag_z of QSOs
	- Redshift

	- psf_g_width
	- psf_r_width
	- psf_z_width
